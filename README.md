A proposta do site é criar um bate-papo com pessoas do futuro. 
Essas pessoas são na verdade a API da IA do Google (Gemini) que foi treinada para assumir determinada personalidade dependendo da escolha do usuário.
A API do Gemini é a única gratuita e possui uma documentação completa para ajudar na hora de implementar a IA no site.
Foi utilizado HTML, CSS e JavaScript, e as animações utilizadas são de CSS puro, sem o uso de frameworks.
